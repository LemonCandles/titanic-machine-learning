# 🚢 TCC: Titanic - Machine Learning from Disaster

> 🚧 **Status do Projeto: Em desenvolvimento** 🚧
>
> Este repositório contém os códigos e experimentos da competição do Titanic no Kaggle, desenvolvidos como parte do meu **Trabalho de Conclusão de Curso (TCC)**.

## 📋 Sobre o Projeto
O objetivo deste projeto é construir um modelo preditivo capaz de responder: *"quais tipos de pessoas tinham maior probabilidade de sobreviver no naufrágio do Titanic?"* utilizando dados de passageiros (idade, sexo, classe, etc).

## 📂 Estrutura do Repositório
A organização inicial das pastas está definida da seguinte forma:

- `data/`: Base de dados original (`train.csv`, `test.csv`) e submissões.
- `notebooks/`: Jupyter Notebooks contendo a Análise Exploratória (EDA) e a modelagem.
- `src/`: Scripts Python auxiliares e funções customizadas.
